# Animacja Chevron

Standardowa animacja obrotu ikony strzałki o 180 stopni, sygnalizująca rozkładanie lub zwijanie elementów interfejsu.

## Podgląd

Klasa `chevron` jest nakładana na element `mat-icon` i współpracuje z klasą `closed`, która jest dodawana, gdy element jest zwinięty.
